# Sts2DebtLoan — 商人借贷 (Merchant Loans)

[English](README.md) · [한국어](README.ko.md) · **中文**

一个 [杀戮尖塔 2](https://store.steampowered.com/app/2868840/) 模组：在商人处**借金币**买下你差一点付不起的东西，然后慢慢还清这笔债——或者被它压垮。

## 核心循环

1. **借款。** 在商店里，付不全的物品也能买下——差额由商人借给你（有上限）。你会获得记录债务的**商人的账本**遗物。
2. **利息增长。** 你要还的比借的多：借款时立即产生的手续费，加上每经过一个房间就累积的利息。拖得越久，代价越高。
3. **拖欠 → 诅咒。** 债务背得太久，会有不断升级的**债务**诅咒牌渗入你的战斗——**逾期**，接着**扣押**，最终是**信用不良**及其无休止的**强制催收**。
4. **偿还。** **定期还款**能力每回合塞给你**还款**牌——打出一张即可花费金币削减本金。**在任意商店偿清本金**即可清除账本、恢复信用——然后你又能再借了。

## 债务商店

一旦欠债，专门的**债务商店**允许你**赊账**购买还款类卡牌，其价格会计入你所欠的债务。货架每次到访都会轮换，且每次都有一张卡打折。

## 还款引擎

你每完成一次还款都会积攒一张**收据**——一种拥有独立计数器的战斗资源。你收集到的还款牌会将收据兑现：

- **还款能力** — *还款优惠*、*返还*、*利息补贴*等会在每次还款时触发，返还格挡、卡牌或金币。
- **消耗收据的卡牌**会随你积攒的收据数量增强：*结算*将其转化为格挡，*账单*转化为多段攻击。
- **催收**把整个循环转为进攻——每回合塞给你一张*强制执行*令牌，消耗一张收据换取**活力**（强化下次攻击）。

## 合作 (多人)

债务是共同承担的负担：

- **传染** — 队友的贷款也会渗入*你的*战斗。
- **人多更狠** — 欠债的玩家越多，利息累积得越快、越高。
- **代偿 (대납)** — 仅限多人的卡牌，替队友偿还一部分债务。当有人错过还款时，最富裕的盟友会得到一张代偿牌，好帮他还债。

## 配置 (游戏内 ModConfig)

最大贷款额，以及商人在哪些幕（Act）提供借贷。

## 状态

积极开发中。已在无画面、无输入的条件下完成端到端验证：**单人**（`solo-verify`）与**双实例合作**（`coop-verify`：商店购买复制、代偿发放、代偿使用均在两端收敛，无不同步）。尚未发布至 Steam 创意工坊。

完整设计说明见 [`DESIGN.md`](DESIGN.md)。

## 构建

本模组是作者 monorepo 的一部分，依赖共享的 **Sts2.ModKit** SDK（csproj 中的 `..\Sts2.ModKit\build\Sts2.ModKit.props`）。若要独立构建，请将该 import 指向一份 Sts2.ModKit 副本。

- **DLL：** `dotnet build Sts2DebtLoan.csproj -c Release` → 部署到 `Slay the Spire 2/mods/Sts2DebtLoan/`。
- **资源包**（遗物/卡牌美术、本地化）：由 `pck_src/` 通过 Godot 4.5.1 `--export-pack` 构建 → `Sts2DebtLoan.pck`。
